네이버 검색 API 버전입니다.

교체 파일:
1. competitor_watch_crawler.py
2. competitor-monitor.yml → .github/workflows/competitor-monitor.yml

Secret:
- 이미 CLIENT_ID / CLIENT_SECRET이 있으면 그대로 작동합니다.
- 권장 이름은 NAVER_CLIENT_ID / NAVER_CLIENT_SECRET 입니다.
- OPENAI_API_KEY는 AI 리포트용입니다.

중요:
네이버 개발자센터 앱에서 '검색' API 사용 권한이 켜져 있어야 합니다.
